API Usage

Overstat is an unofficial Overwatch Stats API written in Go. Accessing its main API endpoint is extremely simple. Perform a GET request on the main lookup endpoint defined below.

For Stats

https://ovrstat.com/{platform}/{username}/profile https://ovrstat.com/{platform}/{username}/complete
